</ol>

# 📖 Project Overview
This project involves:

Arsitektur Data: Merancang Gudang Data Modern Menggunakan Arsitektur Medali dengan Lapisan Perunggu, Perak, dan Emas.
Pipa ETL: Mengekstrak, mentransformasi, dan memuat data dari sistem sumber ke dalam gudang data.
Pemodelan Data: Mengembangkan tabel fakta dan dimensi yang dioptimalkan untuk kueri analitik.
Analitik & Pelaporan: Membuat laporan berbasis SQL dan dasbor untuk wawasan yang dapat ditindaklanjuti. 🎯 Repositori ini adalah sumber daya yang sangat baik bagi para profesional dan siswa yang ingin menunjukkan keahlian dalam:
SQL Development
Data Architect
Data Engineering
ETL Pipeline Developer
Data Modeling
Data Analytics
